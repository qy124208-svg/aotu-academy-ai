============================================
  凹凸学园 · 百日倒数  v5.7
  《凹凸世界》同人校园模拟游戏
============================================

【游戏简介】
你是转校生，在凹凸学园度过一百天。
一百天后——无论发生什么——你都要离开。
18位角色、8对CP、10章主线、9条支线。
每一天，每一个选择，都不可重来。

【打开方式（任选其一）】

方式一：双击 HTML 文件（最简单）
  直接把「凹凸转学记.html」拖到 Chrome/Edge 浏览器窗口即可。

方式二：双击启动脚本（Windows）
  双击「启动游戏.bat」——自动打开浏览器 + 启动本地服务器。

方式三：命令行
  cd 到此文件夹
  python3 -m http.server 8080
  浏览器打开 http://localhost:8080

【系统要求】
  Chrome 80+ / Edge 80+ / 任意现代浏览器
  无需安装任何软件

【文件说明】
  凹凸转学记.html  —— 单文件完整版（可直接打开）
  index.html       —— 骨架（需配合 game.js）
  game.js          —— 游戏主逻辑
  server.js        —— Node.js 本地服务器
  启动游戏.bat     —— Windows 一键启动
